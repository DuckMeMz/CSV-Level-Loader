Generic Hornet App is built on the Hornet engine created by Chris Rook of Northumbria University.
It uses SDL2, STL_image, STL_mixer and STL_ttf.
Licence details, links and further information are available in the docs folder.

Game program written by Martin Wonders

Configuration system written by Daniel Farnham
